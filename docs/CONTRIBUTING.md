# Contributing to Quick-Cart

We appreciate your interest in contributing to the Quick-Cart project! To make the process smooth and effective, please follow these guidelines:

## How to Contribute

1. **Fork the Repository**: Create a personal copy of the repository by forking it.

2. **Create a Branch**: Work on your changes in a separate branch to keep the main branch clean.
   ```bash
   git checkout -b your-branch-name
   ```

3. **Make Your Changes**: Implement your changes, making sure to follow the project's coding style and best practices.

4. **Write Tests**: If applicable, add tests for your changes to ensure they work as expected.

5. **Update Documentation**: If your changes affect the documentation or require new documentation, please update it accordingly.

6. **Commit Your Changes**: Write clear, concise commit messages. Follow the conventional commits style if possible.
   ```bash
   git add .
   git commit -m "feat: add new feature to improve user experience"
   ```

7. **Push Your Branch**: Push your branch to your forked repository.
   ```bash
   git push origin your-branch-name
   ```

8. **Create a Pull Request**: Open a pull request from your forked repository to the original repository. Provide a clear description of your changes and why they are needed.

## Code of Conduct

We adhere to a code of conduct to maintain a welcoming and inclusive environment for everyone. Please follow these principles:

- Be respectful and considerate.
- Use inclusive and non-discriminatory language.
- Provide constructive feedback.

For more details, please refer to our [Code of Conduct](./CODE_OF_CONDUCT.md).

## Reporting Issues

If you find a bug or have a feature request, please follow these steps:

1. **Check Existing Issues**: Ensure that the issue has not already been reported.
2. **Create a New Issue**: Provide a clear description of the issue, including steps to reproduce, expected results, and actual results.

## Contact

For any questions or further discussions, you can contact us through the project's [discussion forum](#) or email us at [support@quick-cart.com](mailto:support@quick-cart.com).
